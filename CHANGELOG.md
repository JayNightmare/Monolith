## [Unreleased]

### Added
- Linked the Monolith name logo on the navbar to direct users back to the landing page.
- User accounts and JWT-based authentication system.
- Secure routes (`ProtectedRoute` client side and `requireAuth` middleware server side) to protect user archives.
- User profile updates on the Settings page (Display Name and Email Address change capability).
- Interactive Pricing page linked directly to MongoDB, enabling plan upgrades/downgrades.
- Enforced account tier limitations:
  - Free Tier: Limited to 3 uploads, basic persona images, restricted access to strategic next moves and recommendations.
  - Pro Tier: Unlimited uploads, high-fidelity photos, and full access to insights.
